# Product Management System

## 📌 Project Overview
Product Management System is a web-based application developed using Spring Boot and Spring MVC. The application allows users to manage product information through CRUD operations with validation, pagination, and a responsive user interface.

## 🚀 Features

- Add New Product
- View All Products
- Update Product Details
- Delete Product
- Product Validation
- Product Category Management
- Pagination Support
- Responsive UI using Bootstrap
- MySQL Database Integration

## 🛠️ Technologies Used

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- Thymeleaf
- MySQL
- Maven
- Bootstrap
- HTML5
- CSS3

## 📂 Project Architecture

- Controller Layer
- Service Layer
- Repository Layer
- Entity Layer
- Database Layer

## 📋 Product Fields

- Product ID
- Product Name
- Product Price
- Product Category
- Product Quantity

## ✅ Validation Implemented

- Product Name cannot be blank
- Product Name must contain at least 5 characters
- Price must be greater than 0
- Quantity must be greater than 0
- Category is mandatory

## 🗄️ Database

Database: MySQL

Table: products

## ⚙️ How To Run

1. Clone the repository

```bash
git clone https://github.com/monali-java-dev/Product-Management-System.git
```

2. Open the project in Eclipse / STS / IntelliJ IDEA

3. Create MySQL database

```sql
create database product;
```

4. Configure database credentials in application.properties

5. Run the Spring Boot application

6. Open browser

```text
http://localhost:8080/
```

## 📸 Modules

### Add Product
Add new products with validation.

### View Products
Display all products in tabular format.

### Update Product
Modify existing product information.

### Delete Product
Remove products from database.

### Pagination
Navigate product records page by page.
